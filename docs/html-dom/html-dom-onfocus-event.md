# HTML | DOM onfocus 事件

> 原文:[https://www.geeksforgeeks.org/html-dom-onfocus-event/](https://www.geeksforgeeks.org/html-dom-onfocus-event/)

**HTML DOM onfocus 事件**支持所有 HTML 标签，除了:
