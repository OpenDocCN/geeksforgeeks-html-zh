# HTML | DOM 窗口默认状态属性

> 原文:[https://www . geesforgeks . org/html-DOM-window-defaultstatus-properties/](https://www.geeksforgeeks.org/html-dom-window-defaultstatus-properties/)

**默认状态**属性用于设置或返回浏览器底部状态栏中的默认文本。

**语法:**

```html
window.defaultStatus
```

**返回类型:**状态栏中的默认文本。

**例:**

```html
<!DOCTYPE html>
<html>

<head>
    <title>
        HTML | DOM Window defaultStatus Properties
    </title>
</head>

<body>

    <p>This is first paragraph.</p>

    <script>
        window.defaultStatus =
            "The default text of the status bar.";
    </script>

</body>

</html>
```

**注意:**

*   The output of this example cannot be generated because this property is not supported by any browser.
*   This property is supported in Opera 12 and earlier versions.
*   This attribute is supported in Firefox 23 and earlier versions.

**支持的浏览器:**主要浏览器不支持 *DOM 窗口默认状态属性*
