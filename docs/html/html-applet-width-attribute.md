# HTML |小程序宽度属性

> 原文:[https://www.geeksforgeeks.org/html-applet-width-attribute/](https://www.geeksforgeeks.org/html-applet-width-attribute/)

**HTML 小程序宽度属性**用于指定<小程序>元素的宽度。

**语法:**

```html
<applet width="pixels">
```

**属性值:**
**像素:**它包含指定小程序元素宽度的像素值。

**示例:**

```html
<!DOCTYPE html>
<html>

<head>
    <title>
        HTML | applet width attribute
    </title>
</head>

<body>
    <applet code="HelloWorld" 
        width=200 height=60>
    </applet>

    <h2>
        Hello GeeksForGeeks
    </h2>
</body>

</html>
```
