**HTML 标签:** HTML 标签是隐藏的关键词，用于创建不同格式的网页。大多数标签包含两部分，开始标签和结束标签。开始和结束标记的文本相同，但结束标记包含正斜杠(/)字符。是一些不需要结束标记的标记。
