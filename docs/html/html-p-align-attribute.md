对齐属性

> 原文:[https://www.geeksforgeeks.org/html-p-align-attribute/](https://www.geeksforgeeks.org/html-p-align-attribute/)

**HTML < p >对齐属性**用于*指定段落文本内容的对齐方式*。

**语法:**

```html
<p align="left | right | center | justify">
```

**属性值:**

*   **左:**设置文本左对齐。
*   **右:**设置文本右对齐。
*   **居中:**设置文本居中对齐。
*   **对齐:**拉伸段落文本，使所有行的宽度相等。

**注意:**HTML 5 不支持< p >对齐属性。

**示例:**

```html
<!DOCTYPE html>
<html>

<head>
    <title>
        HTML p align Attribute
    </title>
</head>

<body>
    <h1>GeeksforGeeks</h1>

    <h2>HTML p align Attribute</h2>

    <p align="left">
      Left align content
  </p>
    <p align="center">
      center align content
  </p>
    <p align="right">
      Right align content
  </p>
</body>

</html>
```
