# HTML | onkeypress 属性

> 原文:[https://www.geeksforgeeks.org/html-onkeypress-attribute/](https://www.geeksforgeeks.org/html-onkeypress-attribute/)

当用户按下键盘上的某个键时，此属性会触发。此事件属性不能用于所有浏览器中的所有按键(如 ALT、CTRL、SHIFT、ESC)。
